User Interface
Figure 7. Login
Figure X presents the login interface of the Maramag Fish Landing Information System. The interface uses a split-screen layout to create a clean, professional, and user-friendly first impression. The left panel contains the main login components, including the email field, password field, “Remember me” option, “Forgot your password?” link, and the Sign In button. These elements are arranged clearly so users can easily understand how to access the system. The right panel highlights the identity of the system by showing the Maramag Fish Landing logo and the text MAANYAG MARAMAG!, which strengthens the branding and visual recognition of the application. The color scheme uses deep blue, white, and gold accents to give the interface a formal yet welcoming appearance. Overall, the login page was designed to be simple, organized, and easy to use for both staff and broker users.
